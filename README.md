# Kami Inventory (Offline-first)

אפליקציית ווב פשוטה לניהול ציוד, סטים והשאלות — עובדת מקומית עם SQLite וללא תלות בענן.

## יכולות עיקריות
- ניהול משתמשים (הרשמה/התחברות).
- ניהול סטים (kits) ופריטי ציוד (items).
- תמיכה בפריטי ציוד שלא משויכים לסט (ציוד אקסטרא/מיוחד).
- תהליך השאלה על סט (חתימה) + אפשרות לצרף פריטים בודדים.
- תהליך זיכוי/החזרה עם צ'קליסט למצאי בפועל.
- לכל משתמש מסך "הציוד שעל שמי" (השאלות פעילות).

## התקנה והרצה (מחשב ללא אינטרנט)

1. התקינו Python 3.10+ על המחשב היעד.
2. העתיקו את תיקיית הפרויקט בשלמותה.
3. צרו סביבה וירטואלית והתקינו תלויות:

```bash
python -m venv .venv
source .venv/bin/activate  # ב-Windows: .venv\\Scripts\\activate
pip install -r requirements.txt
```

4. אתחול בסיס נתונים:

```bash
python -c "from app import init_db; init_db()"
```

5. הרצה:

```bash
python app.py
```

6. פתחו בדפדפן: http://127.0.0.1:5000

## הערות
- כל המידע נשמר בקובץ מקומי `inventory.db`.
- ניתן לבצע גיבוי פשוט על ידי העתקת `inventory.db`.
